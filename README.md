# 📰 News Management Platform

A full-stack news management system with a powerful backend and admin dashboard for real-time content control and delivery.

## 🚀 Features

- 📰 Create, update, delete, and manage news
- 🔥 Breaking news with instant push notifications
- 👤 Anonymous & authenticated user support
- 🔐 Role-based access control
- 📊 Install / uninstall tracking for analytics
- 💰 Google AdMob integration for monetization
- 📡 Real-time updates and notifications
- 🧑‍💻 Admin dashboard for full control

## 🏗️ Tech Stack

- Backend: Node.js / Express (or your stack)
- Database: MongoDB / PostgreSQL
- Frontend: React (Admin Dashboard)
- Notifications: Push Notification Service (FCM or similar)
- Deployment: AWS
- CI/CD: GitHub Actions

## ☁️ Deployment

The project is deployed on **AWS** with automated CI/CD pipeline using **GitHub Actions** for smooth build and deployment flow.
